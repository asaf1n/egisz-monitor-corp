# EGISZ Corp — Power BI Project (PBIP)

Каталог содержит **файлы проекта Power BI** (`*.pbip` + артефакты `*.SemanticModel` / `*.Report`), а также **SQL-запросы** из провижининга Metabase (`sql/`) и дополнительные отчётные запросы (10–13).

Metabase в кластере **не заменяется**; используется **тот же** PostgreSQL. См. [docs/POWERBI.md](../../docs/POWERBI.md).

## Открытие в Power BI Desktop

1. Установите актуальную [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
2. **Файл → Открыть → Обзор** и выберите файл **`egisz-corp.pbip`** в этом каталоге.
3. Если проект не открывается (несовпадение версии схемы Fabric / отсутствие встроенной темы): создайте пустой отчёт, **Сохранить как → Файлы проекта Power BI** в этот же каталог с именем `egisz-corp`, затем перенесите папку `sql/` и документацию в объединённый каталог вручную, либо подключите модель к данным заново по [docs/POWERBI.md](../../docs/POWERBI.md).

## Подключение витрины PostgreSQL

1. **Преобразовать данные** → **Создать запрос** → **База данных PostgreSQL** (или расширьте модель: **Главная → Управление данными → Параметры источника данных**).
2. Укажите сервер и базу (локально: `localhost` и порт из Docker / port-forward; см. корневой `README.md` и `k8s/README.md`).
3. Для каждой страницы отчёта (01–09) создайте запрос с **нативным SQL**: скопируйте блок из соответствующего файла `sql/NN_*.sql` (в одном файле несколько карточек — разделяйте на отдельные запросы в Power Query).

Рекомендуемые представления:

- `public.v_egisz_transactions_enriched_ui`
- `public.v_rpt_documents_no_response_ui`
- `public.stg_parse_errors`
- `public.etl_state`

## Семантическая модель в репозитории

`egisz_corp.SemanticModel` содержит таблицу-заглушку **ReadmePlaceholder** (одна строка-подсказка). После подключения к Postgres замените источники на реальные запросы или **Удалите** таблицу-заглушку и импортируйте витрину через Power Query.

## Кэш Power BI в git

Локальные артефакты в `.pbi/` игнорируются корневым `.gitignore`. Не коммитьте пароли в параметрах проекта.
